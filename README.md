# 🚑 PulseIQ – AI-Powered Emergency Care Resource Allocator

## 📌 Project Overview

PulseIQ is an AI-powered emergency healthcare resource allocation system that helps patients quickly identify the most suitable hospital during medical emergencies. The platform recommends hospitals based on emergency type and available resources such as beds, ICU capacity, ambulances, and specialist availability.

---

## 🎯 Problem Statement

During emergencies, patients often spend valuable time searching for hospitals with available resources. This delay can negatively affect treatment outcomes. Hospitals also face challenges in efficiently managing emergency resources.

---

## 💡 Our Solution

PulseIQ uses AI-assisted decision support to recommend the most appropriate hospital based on:

* Emergency type
* Available beds
* ICU availability
* Ambulance availability
* Specialist availability

The platform also generates QR-enabled PDF reports, provides an analytics dashboard, and displays hospital locations on an interactive map.

## 🚀 How to Run

1. Clone the repository.
2. Install dependencies:

```bash
pip install -r requirements.txt
```

3. Run the application:

```bash
streamlit run app.py
```
